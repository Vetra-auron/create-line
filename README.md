# create-line

데스크톱에서 실행되는 거래처 검색 및 메모 관리 앱입니다. 엑셀 파일로부터 거래처 정보를 불러와 대표자명 또는 거래처명을 기반으로 검색하고, 선택한 거래처에 대한 특이사항을 기록할 수 있습니다.

## 준비 사항

1. Python 3.10 이상을 설치합니다.
2. 필요한 패키지를 설치합니다.

```bash
pip install -r requirements.txt
```

## 실행 방법

1. `app.py`를 실행합니다. 엑셀 파일 경로를 인자로 전달하면 앱이 자동으로 파일을 불러옵니다.

```bash
python app.py [엑셀_파일경로.xlsx]
```

2. 실행 후 "엑셀 불러오기" 버튼을 눌러 파일을 선택하거나, 시작할 때 인자로 넘겨준 파일을 사용합니다.
3. 검색창에 대표자명 또는 거래처명을 입력하고 `Enter` 키를 누르거나 "검색" 버튼을 누르면 결과가 목록으로 표시됩니다.
4. 목록에서 거래처를 선택하면 상세 정보가 오른쪽 패널에 표시되고, 하단에서 특이사항을 작성하여 "특이사항 저장" 버튼으로 저장할 수 있습니다. 특이사항은 `notes.json` 파일에 저장됩니다.

## 참고 사항

- 엑셀 파일의 열 이름은 `거래처명`, `대표자명` 등이 권장되지만, 앱은 문자열이 포함된 모든 열을 대상으로 검색을 수행합니다.
- 특이사항은 선택된 거래처의 고유 값(거래처 코드, 사업자번호 등)을 기준으로 저장되며, 해당 정보가 없는 경우 목록에서 표시되는 항목과 인덱스를 조합하여 저장합니다.
